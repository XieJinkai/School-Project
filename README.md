软件开发：使用 C# 、 .NET Framework 4.7.2 与 WinForms 搭配 SunnyUI 构建客户端界面，承载活动申请、查询、支付与用户登录注册

数据库管理：采用 SQLite 本地文件数据库进行数据存储与检索

支付集成：调用支付宝沙盒环境，成功模拟实现活动报名的支付功能

语音控制：调用系统库API实现语音识别功能，通过语音指令实现界面切换

串口通信：通过串口通信实现条码扫描器录入信息，扫码签到和付款功能
