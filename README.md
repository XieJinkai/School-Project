软件开发：在Winform框架下设计用户界面，实现C/S端设计。通过多线程、TCPSocket网络编程和MySQL数据库实现在线活动查询、活动报名、活动申请等多功能。
数据库管理：使用MySQL实现用户信息、活动信息的录入与展示，确保数据存储的准确性和高效性。

语音控制：调用系统库API实现语音识别功能，通过语音指令实现界面切换，提升用户体验和操作效率。
串口通信：通过串口通信实现条码扫描器录入信息，提高了信息录入效率和准确性。
